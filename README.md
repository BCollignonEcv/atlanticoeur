# Atlanticoeur

https://bcollignonecv.github.io/atlanticoeur/

## Setup 

### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Run API - Json server
```
json-server server/db.json --routes server/routes.json
```

#### Compiles and minifies for production
```
npm run build
serve -s dist
```

#### Lints and fixes files
```
npm run lint
```
